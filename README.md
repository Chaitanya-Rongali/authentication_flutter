## Getting Started:
#### Prerequisites:
- List required tools:
- Flutter SDK (stable version)
- Dart SDK
- Android Studio / VS Code (recommended IDE)
### 1. Clone the repository

```bash
git clone https://github.com/Chaitanya-Rongali/authentication_flutter.git
```
### 2.Navigate to the project directory:
```bash
cd authentication_flutter
```
### 3.Get dependencies:
```bash
flutter pub get
```
### 4.Run the app:
```bash
flutter run
```
